什么是 Clash？

Clash 是一款开源的基于规则的代理工具，广泛应用于科学上网领域。它支持多种代理协议，包括 Shadowsocks、Vmess、Trojan 等，以及更多的自定义协议，帮助用户突破网络封锁，保护上网隐私。通过灵活的规则配置，用户可以根据不同需求，选择不同的代理服务器或线路，提高网络速度，优化上网体验。

Clash 提供了非常强大的功能，用户不仅可以选择不同的代理协议，还可以通过配置文件或订阅链接的方式，动态管理和更新代理服务器的列表。它支持跨平台使用，覆盖 Windows、Mac、Linux、Android 和 iOS 等多种操作系统，确保用户在各种设备上都能获得稳定、快速的网络连接。

什么是 Clash 订阅链接？

Clash 订阅链接是一种特殊的 URL，通过这个链接，用户可以自动导入并更新其代理配置。当用户订阅某个代理服务时，服务商会提供一个订阅链接，用户可以将这个链接导入到 Clash 客户端中。Clash 会定期从该链接获取最新的代理配置，无需手动干预，就能够自动更新节点信息或规则。

通过订阅链接，用户不仅能够轻松切换代理服务器，还能随时获取服务商的最新配置，确保网络连接始终保持最佳状态。无论是切换服务器节点，还是调整代理规则，订阅链接都能让整个过程变得更加高效和便捷。

FAQ

VMess 节点的 alterId 参数还需要设置吗？
在新版 V2Ray/Xray 内核中，alterId 已被废弃，不再影响认证逻辑。建议用户设置为 0 或删除该字段，以避免连接异常或不兼容情况。
如何用 iptables 限制 V2Ray 服务的访问来源？
使用 iptables 规则仅允许可信 IP 或 CDN 的中转 IP 访问服务端口，例如 `iptables -A INPUT -p tcp -s --dport 443 -j ACCEPT`，并在默认策略下拒绝其他来源，从而降低被扫描与滥用风险。
VLESS 在高封锁环境下的优势是什么？
VLESS 去掉了内置加密，依赖 TLS 层安全，减少被 DPI 检测的特征。结合 XTLS 或 Reality 模式，可以更好地隐藏流量，提高在高封锁网络中的稳定性与隐蔽性。
V2Ray 如何隐藏真实 IP？
通过启用 TLS、Reality 或 WebSocket + CDN 转发，V2Ray 可有效隐藏真实服务器 IP。这样外部监测只能看到中转节点，从而提升匿名性与安全性。
QuantumultX 与 Shadowrocket 相比的最大区别是什么？
QuantumultX 在规则系统和自动化脚本上更灵活，支持丰富的 rewrite 与 filter 功能，而 Shadowrocket 则更轻便易用，适合普通用户快速切换节点。二者可根据使用习惯与功能需求选择。
Surfboard 与 Shadowrocket 有什么区别？
两者都是 iOS 上的代理工具。Shadowrocket 功能更全面、更新更频繁，而 Surfboard 以轻量和快速见长。若只需简单分流与节点切换，Surfboard 足以满足日常使用。
Hysteria2 多路复用如何优化短连接性能？
多路复用允许多个短连接共享同一通道，减少握手延迟和资源消耗。适用于游戏、即时通讯等高频短连接场景，提高吞吐量和连接稳定性，优化整体网络体验。
Hysteria2 的 QUIC 模式相比 TCP 有哪些优势？
QUIC 协议基于 UDP，可减少握手延迟并在高丢包环境下保持较好体验。Hysteria2 对 QUIC 进行了优化，支持自适应拥塞控制与加密，能在移动网络中比 TCP 拥有更快的响应速度。
如何检测并处理订阅中的死节点？
定期运行批量测速脚本或使用客户端内置的 url-test、health-check 功能，自动标记或移除响应超时或重连失败的节点。结合订阅更新策略可自动替换或提醒用户维护节点列表。
Potatso Lite 可以管理多少个节点？
Potatso Lite 可以管理数十个节点，适合轻量用户使用。虽然不支持批量自动更新，但可以手动添加和删除节点，结合策略组实现基本分流和管理功能。
